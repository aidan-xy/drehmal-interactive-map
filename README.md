An interactive map for the Minecraft adventure map Drehmal.

Currently, the only step left is to populate the map with markers. The only marker category that is
fully mapped are Terminus Towers. To contribute markers, see `./src/markers/ExampleMarkers.jsx`.

To run for yourself, ensure you have an up to date npm installation, clone the repo, then run the 
commands:
1. `npm install`
2. `npm run build`
3. `npm run dev`

from your terminal in the root directory. Visit the shown localhost link in any web browser.
